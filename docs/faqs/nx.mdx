# NX and Onxy

## Create a new Onyx application

### Question

How do I create a new Onyx application?

### Answer

`pnpm nx workspace-generator onyx-app --name=name-of-the-app`

## Create a new Onyx Storybook library

### Question

How do I create a new Onyx Storybook library?

### Answer

`pnpm nx generate @nrwl/react:lib test-react-lib`

`pnpm nx generate @nrwl/react:storybook-configuration test-storybook-config`
