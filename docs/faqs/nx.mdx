# NX and Onxy

## Create a new Next.js application

### Question

How do I create a new Next.js application?

### Answer

`pnpm nx g @nrwl/next:app my-new-app`

## Create a new Storybook library

### Question

How do I create a new Storybook library?

### Answer

`pnpm nx generate @nrwl/react:lib test-react-lib`

`pnpm nx generate @nrwl/react:storybook-configuration test-storybook-config`
